# 🚢 Морской Бой

Классическая игра "Морской бой" с искусственным интеллектом, написанная на Python с использованием PyQt6.


## 🎮 Особенности

- **Умный ИИ противник** - продвинутый алгоритм с двумя режимами: случайные выстрелы и режим охоты
- **Полная система статистики** - отслеживание побед и поражений в базе данных SQLite
- **Красивый визуал** - анимации огня на потопленных кораблях, кастомные стили оформления
- **Несколько окон** - главная игра, детальная статистика, информация о программе, настройки
- **Интуитивное управление** - простой и понятный интерфейс

## 🎯 Игровой процесс

### Расстановка кораблей
- 1 корабль на 4 клетки
- 2 корабля на 3 клетки  
- 3 корабля на 2 клетки
- 4 корабля на 1 клетку

### Управление
- **Клик по своему полю** - расстановка кораблей (в фазе размещения)
- **Клик по полю противника** - совершение выстрела
- **Кнопка "Повернуть корабль"** - изменение ориентации корабля (горизонтально/вертикально)
- **Кнопка "Статистика"** - просмотр детальной статистики игр
- **Кнопка "О программе"** - информация о разработчике
- **Кнопка "Настройки"** - меню настроек (в разработке)
- **Кнопка "Начать заново"** - перезапуск игры

## 🛠 Технологии

- **Python 3.8+** - основной язык программирования
- **PyQt6** - фреймворк для графического интерфейса
- **SQLite3** - база данных для хранения статистики
- **Random** - алгоритмы искусственного интеллекта
- **Datetime** - временные метки в статистике

🤖 Искусственный интеллект
ИИ использует сложную двухрежимную систему:

Режим случайных выстрелов
Выбирает случайные непрострелянные клетки

Используется пока не было попаданий

Режим охоты (Hunting Mode)
Активируется после первого попадания

Ищет корабль в четырех направлениях от точки попадания

Определяет ориентацию корабля при втором попадании

Добивает корабль вдоль определенной ориентации

💾 Система статистики
Автоматическое сохранение результатов каждой игры

Детальная статистика - количество побед, поражений, процент побед

База данных SQLite - данные хранятся в файле battleship.db

Временные метки - запись даты и времени каждой игры
